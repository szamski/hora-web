This repository hosts the public website for **hora Calendar** — a native macOS Google Calendar client built with Swift 6 and SwiftUI.

### What's here

- `index.html` — Landing page, privacy policy, and terms of service
- `assets/` — App icon, demo GIF, Bumbbled font
- `.github/workflows/deploy.yml` — Auto-deploy to GitHub Pages on push
